\SUB{1\ \ 引言}%一级标题
价格预测是永恒的话题, 市场变幻莫测, 即使在一段时间可以捕捉到先行指标进行较为准确的预测, 但是这样的套利机会很快会被市场消化.\\

我国的鸡蛋期货于2013年11月8日在大连商品交易所推出, 目前关于鸡蛋期货的研究还很少. 美国曾经推出鸡蛋期货, 但是随着鸡蛋的价格趋于稳定, 套保需求大幅下降. Miracle(1972)通过对美国鸡蛋期货市场1940-1966的研究发现套期保值交易的数量占期货市场交易的大部分, 并且提出价格波动的减小会导致在鸡蛋期货市场的投机者流失,进而更加降低了套保需求. 美国鸡蛋期货市场交易量在1961年后开始呈现下滑趋势, 最终CME于1982年9月停止了鸡蛋期货交易. 中国的鸡蛋期货于推出不到一年, 交易活跃, 年化波动率最高时近$30\%$. 不排除在若干年后我国的鸡蛋期货市场因为鸡蛋价格变得可预测等原因而交易减少的可能, 就目前而言, 我国鸡蛋期货市场发展良好. 因此, 借鉴曾经基于美国对鸡蛋期货市场的研究成果, 对我国鸡蛋期货产品进行价格预测, 对于鸡蛋期货市场的套期保值者和投机者, 是十分必要的.\\

Fama和French(1987)把关于大宗商品期货价格的较为流行的研究观点分为两类: 第一类认为即期价格和期货价格的差异是由于用于存储大宗商品的本金利息损失、仓储成本和存货的便利收益率.第二类观点把期货价格分割为期望的风险溢价和期货的即期价格的预测进行研究.\\

差分自回归移动平均模型(Autoregressive Integrated Moving Average Model, 简记为ARIMA), 是Box和\\Jenkins于70年代初提出的时间序列预测方法. ARIMA模型已经被用于大宗商品价格的预测, 如Weiss(2000), Chinn, LeBlanc, Coibion(2001)和Buchanan, Hodges, Theis(2001), 均得到较好的结果. 但是Bessler, David, Brandt(1987)经研究认为ARIMA 模型关于畜牧类大宗商品价格不能提供好的预测.\\

本文基于第二类观点, 通过ARIMA模型充分提取已有的鸡蛋期货产品的每日市场价格的信息进行提前一天的预测. 不同于以往的ARIMA模型的应用, 在进行时间序列分析之前, 首先对市场中其他可能提高预测精度的变量进行Granger因果关系检验, 并对因变量关于该变量进行处理.  \\

\SUB{2\ \ ARIMA(p,d,q)模型}%一级标题
记$y_t$为期货产品在t时刻的价格, 根据Box, Jenkins, Reinsel(2013) 其一般的ARIMA模型如下:
\begin{equation}
\phi(B)(1-B)^d y_t=\theta(B)\varepsilon_t
\end{equation}
其中$\phi(B)=1-\sum_{i=1}^{p}\phi_i B^i,\theta(B)=1-\sum_{j=1}^{q}\theta_j B^j,$ B是滞后算子: $B^ky_t=y_{t-k}$, $\varepsilon_t$是独立同分布的随机变量序列, 满足$E(\varepsilon_t)=0, Var(\varepsilon_t)=\sigma_{\varepsilon}^2>0.$
\\ 基于上述模型, 我们作了以下两种应用：\\
A: 关于$y_t$的对数收益率进行拟合
\begin{equation}
\phi(B)(1-B)^d ln\left(\frac{y_t}{y_{t-1}}\right)=\theta(B)\varepsilon_t
\end{equation}
B: 引入与$y_t$相关的变量$x_t$
\begin{equation}
\phi(B)(1-B)^d ln(y_t-\alpha x_{t-1})=\theta(B)\varepsilon_t
\end{equation}
其中, $\alpha$ 为常数.\\

\SUB{3\ \ 实证}%一级标题

大宗商品的即期价格普遍被认为难以获得或者难以直接观测. 通过鸡蛋的市场价格在直观上是合理的想法, 但是地域价格差异和不同市场的价格差异都是较难处理的因素, 基于此, 本文使用的数据均来自期货市场(大连商品交易所), 且主要研究对象为交易量较大的期货产品,交易量和成交金额较小的产品不在考虑范围内. 另外, 避免特殊性，选取当日最高价和最低价的平均值为研究对象。因为鸡蛋期货上市时间不到一年, 数据有限, 所选用的数据为每日数据. 时间序列分析时暂不考虑季节因素.\\

\sub{3.1\ 基于ARIMA模型的JD1409价格的动态预测\ }
便于比较，首先直接针对JD1409用ARIMA模型进行预测.为充分利用已有价格信息,减小误差, 取日对数收益率
\begin{equation}
u_t=ln\left(\frac{y_t}{y_{t-1}}\right)
\end{equation}
为变量进行时间序列分析. 选取对数收益率为研究对象限制了模型只能进行提前一天的预测, 但是这刚好与引入ML1409的ARIMA模型也只能进行提前一天的预测相对应.\\

\sub{3.2\ Granger因果关系检验\ }

从成本因素看，鸡蛋价格主要受鸡苗, 玉米和豆粕影响,据农业部统计玉米占鸡蛋生产成本的$42\%-46\%$, 与鸡蛋的相关性系数为0.74. 豆粕占鸡蛋成本的$14\%-17\%$, 与鸡蛋的相关系数为0.59. 豆粕期货和玉米期货产品分别在2000年和2004年已经上市,是较成熟的期货产品. 但是我们发现对于同时间到期的期货产品, 鸡蛋期货产品的价格与豆粕期货产品价格有更高的相关性. 所以期货下文先从豆粕期货板块中选择与鸡蛋期货存在因果关系的期货产品. 虽然经济意义上豆粕价格影响鸡蛋的成本价格,避免主观性, 利用Granger因果关系检验来分析豆粕具体的期货产品和鸡蛋期货产品的因果关系. Granger因果关系检验之前首先要检验时间序列的平稳性. 通过Augmented DickeyCFuller(ADF)检验来判断是否存在单位根, 从而判别其平稳性. \\
原假设：$H_0 :\rho_0=1$,即序列存在单位根.\\
备择假设：$H_1 :|\rho_0|< 1$,即序列不存在单位根。\\
当$H_0$为真的概率$p$小于显著性水平0.05时, 拒接原假设. 分析鸡蛋期货JD1409合约,JD1405 合约, 豆粕期货ML1405合约, ML1409合约的滞后差分项时间序列，ADF 检验结果值均为$p=0.01$, 小于给定显著性水平，因此拒绝原假设,认为JD1409合约,JD1405合约, 豆粕期货ML1405合约, ML1409合约的一阶滞后差分项是平稳的,能进行Granger因果关系检验.\\

Granger因果关系检验的基本思想是检验在给定经济领域中两个时间序列${Y_t}$ 和${X_t}$ 之间是否存在预测关系. 如果$X$的历史数据有助于预测$Y$ 的当前值,那么$x$是$y$的Granger原因(Granger 1969). \\
建立回归模型如下:\\
\begin{equation}
y_t=c_1+\sum_{i=1}^{p}\alpha_i y_{t-i}+\sum_{j=1}^{q}\beta_j x_{t-j}+\varepsilon_t,t=1,2,\cdots,n
\end{equation}
与\\
\begin{equation}
y_t=c_1+\sum_{i=1}^{p}\alpha_i y_{t-i}+\varepsilon_t, t=1,2,\cdots,n
\end{equation}
原假设：$H_0 :\beta_1=\beta_2=\cdots=\beta_q=0$,即$x$不是$y$的Granger 原因.\\
备择假设：$H_1 :\beta_j\neq 0(j=1,2,\cdots,q)$\\

检验统计量$E=\frac{[RSS(p)-RSS(q,p)]q^{-1}}{RSS(q,p)(n-p-q-1)^{-1}}~F(q,n-p-q-1)$，其中$RSS(p), RSS(q,p), n$分别为两个模型的普通最小二乘估计的残差平方和以及样本容量. 对鸡蛋期货JD1409合约(下文简称JD1409)和JD1405 合约(下文简称JD1405), 豆粕期货ML1405合约(下文简称ML1409)和ML1409 合约(下文简称ML1409)进行时间序列的单向Granger因果关系检验, 结果如表1所示.
\begin{center}{\sz

\hline

\sub{3.3\ 基于ARIMA模型引入ML1409后的JD1409动态预测\ }

记 $\{y_t\}$是由JD1409从2013年11月8日到2014年5月23日的价格构成的时间序列, $t=1,2,\cdots, 132$. $\{x_t\}$是由ML1409从2013年11月8 日到2014年5月23 日的价格构成的时间序列, $t=1,2,\cdots, 132$. 在3.1中最优的均为ARIMA(2,1,1), 推测$\{y_t\}$具有二级自回归性; 并且在R环境中对${y_t}$进行ACF分析显示其对一阶和二阶滞后项的回归系数都较显著, 因此建立回归模型如下：
\begin{equation}
y_t=\alpha x_{t-1}+\beta_1 y_{t-1}+\beta_2 y_{t-2}+\varepsilon_t, t=1,2,\cdots,n
\end{equation}
通过线性回归最小二乘获得线性拟合模型从而获得对应的$\alpha$. 取不同的n值, 将会得不到不同的$\alpha$ 值,下文中将对$\{y_t-\alpha x_{t-1}\}$ 进行时间序列分析.\\
由求得的$\alpha$, 对给定的n, 对$\{y_t-\alpha x_{t-1}\},t=1,2,\cdots,n$进行时间序列分析, 在R环境中选择最优的ARIMA模型.

\SUB{4\ \ 结论}%一级标题

文章提出了可动态预测期货价格的方法, 并针对鸡蛋期货产品JD1409进行验证. 不同于以往的ARIMA\\
模型的应用, 首先对市场中可能提高预测精度的变量进行Granger 因果关系检验, 并对因变量处理后进行时间序列分析. 实证结果显示, 引入具有Granger因果关系的变量后研究对象的预测准确度明显增强,并且预测值的$80\%$的置信区间更小. 在以后的研究中, 还可以从更广的范围寻找变量进行Granger因果关系检验, 优化对因变量的处理, 提高拟合度和预测精度.\\
